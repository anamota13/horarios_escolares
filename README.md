# Horários Escolares

Aplicativo simples desenvolvido em Flutter para gerenciar horários escolares.

## Funcionalidades

- Adicionar horários  
- Listar horários  
- Excluir horários  

## Tecnologias

- Flutter  
- Dart  
- Material Design 3
